# Sistema Interno de Inventario y Venta de Bebidas Gaseosas

Sistema web interno para control de inventario, ventas, compras, caja y rentabilidad.

## Tecnologías

- **Astro 5** - Framework web
- **HTMX** - Interacciones parciales
- **Alpine.js** - Interactividad local
- **SQLite/Turso** - Base de datos
- **Biome** - Linter y formatter
- **pnpm** - Gestor de paquetes

## Requisitos

- Node.js >= 20.0.0
- pnpm >= 9.15.0

## Instalación

pnpm install
